Conceitos extras - HTML e CSS

Link "mailto" - O link Mailto é um tipo de link HTML que ativa o cliente de email padrão no computador para enviar um email. O navegador da web requer um software cliente de email padrão instalado em seu computador para ativar o cliente de email. Se você possui o Microsoft Outlook, por exemplo, como seu cliente de email padrão, pressionar um link mailto abrirá uma nova janela de email.

*Referência: W3C Schools.com

API Fontes do Google - Você não precisa fazer nenhuma programação; tudo o que você precisa fazer é adicionar um link de folha de estilo especial ao seu documento HTML e, em seguida, consultar a fonte em um estilo CSS.

*Referência: Google Fonts - Guia

Propriedade CSS: Cursor - A propriedade CSS cursor especifica o cursor do mouse mostrado quando o ponteiro do mouse está sobre um elemento. Em nosso site foi usado o 'cursor:pointer', isto é, quando o cursor está sobre links ele mostra uma mão, a qual serve para informar ao usuário que aquilo é um link (Fotos dos integrantes são links).

*Referência: Mozilla Developer - Cursor

Propriedade CSS: list-style-type - A propriedade CSS list-style-type pode ser aplicada a qualquer elemento cujo valor 'display' esteja definido como 'list-item'. Além disso, como essa propriedade é herdada, ela pode ser configurada em um elemento pai (geralmente <ol> ou <ul>) para aplicá-la a todos os itens da lista.

*Referência: Mozilla Developer - List-style-type
