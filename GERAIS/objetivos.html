Objetivos da Disciplina

O grupo pretende colocar em perspectiva a programação para Web. Bem como, nos capacitar para compreender as principais linguagens de marcação e estilo. Ter a capacidade de compreender e desenvolver scripts para execução em navegadores. Por fim, temos por objetivo concluir todos os requisitos da disciplina sendo capaz de compreender e desenvolver scripts para execução em servidores web.

Entender conceitos fundamentais sobre:

  • HTML
  • CSS
  • JavaScript
  • PHP

O grupo ficaria muito mais feliz se não houvessem reprovações de nenhum dos integrantes na disciplina!
